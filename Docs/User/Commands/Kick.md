# /Kick

## Basic Information

#### Description
Remove a client from the server.

#### Aliases
- k

#### Usages
- /kick \<clientId>

#### Arguments
- \<clientId> - Client to kick.

#### IsHostOnly?
- True

#### CanSpectatorRun?
- True

## More Information
The kick command is a host only command which allows hosts to remove clients given their id. It is the only way for a 
host to remove a spectator.
